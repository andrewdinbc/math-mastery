# Mastery Studio
(repo name math-mastery pending rename — see docs/PROJECT_SPEC.md)

Keller Model / PSI (Personalized System of Instruction) applied to math:
self-paced, mastery-gated micro-units with AI as proctor — automatic
marking plus targeted "chalkboard style" remediation on specific mistakes.

Three integrated purposes:
1. **Mastery practice** — self-paced, differentiable mastery thresholds per student.
2. **QR/Lesson Planner delivery** — micro-units authored in Lesson Planner, delivered via the same QR-code worksheet pattern as Student Portfolio (print, blank+scan, or online).
3. **Teacher assessment analytics** — CoGrader-style holistic strength/deficiency breakdown, shared with assessment-tool.
